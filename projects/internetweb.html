Internet

A global computer networks that allows to give information from different computers from across the globe.

Web

A system of interconnected computers that has documents and resources that could be access through internet or network of computers.

How are the Internet and Web different?

The internet refers to the networks of computers while the web is the system that is using the internet to share information.

ASCII and Unicode

ASCII stands for American Standard Code for Information Interchange. This character encoding is only limited to american letters, punctuations, and numbers. While Unicode is a character encoding is universal, meaning you can use all of the characters that are available.

What is UTF-8

This is a character encoding that is mainly used in creating websites. This is also part of the Unicode character encoding.

URL

This means Uniform Resource Locator. Used to locate the website on the internet for example, www.google.com

Domain Name

This is part of the URL where it allows the user to what website specifically he or she wants to open. For example, in www.google.com, the 'google' is the domain name.

HTTP

Stands for Hypertext Transfer Protocol. This is used to transfer all kinds of files between a computer and a server.

HTTPS

Stands for Hypertext Transfer Protocol Secure, this is the more secure version of the HTTP which is possible through file encryption between a computer and a server.

IP Address

Consists of group of numbers separated by decimals, generally used to identify a specific computer base on what IP address that computer is assigned to in a network.

DNS

Stands for Domain Name System. This translates the common URL like google.com into set of numbers for computers to recognize and read.

W3C Recommendations

Standard guidlines by the World Wide Web Consortum (W3C) to make sure that webpages are working properly.

XML, XHTML, HTML, HTML5

Extensive Markup Language (XML), Extensive Hypertext Markup Language (XHTML), Hypertext Markup Language (HTML), Hypertext Markup Language 5, or the fifth version of the HTML (HTML5)
