Engagement Question:

1. When should you use a backgroung image, and when should you use the IMG tag?

We use the background image in CSS when the image is in pattern or decorative, when the image needs to be in a specific position and that you want to do it flexibly. On the other hand, we use the IMG tag whenever the image is content-related like a product photo, diagrams, etc. We also use the IMG tag if we want the photo to be clickable in the webpage.
