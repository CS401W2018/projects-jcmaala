Engagement Questions:

1. The form tag has an "action" attribute. What does it do?

The action attribute in HTML specifies the URL where the form data should be sent when the user submits the form.

2. The form tag has a "method" attribute. What does it do?

The method attribute in HTML specifies how the form data should be sent to the server when the form is submitted.

3. What is the difference between a "POST" and a "GET"?

The "GET" is used for retrieving or filtering data like search forms, links, etc. while "POST" is used for sensitive data or actions that modify data like login, registration, and form submissions.

Engagement Questions (added)

1. What is JavaScript? How did it get its name?

JavaScript is a programming language primarily used for making web pages interactive. JavaScript was created by Brendan Eich at Netscape which they called it Mocha first and then LiveScript. When Netscape joined the Sun Microsystems later on, they renamed it JavaScript to ride the popularity of Java at that time.

2. What are two ways you can run JavaScript?

It can run directly inside web browsers like Chrome, Firefox, Edge, and Safari. It is embedded in HTML pages. It can also run on a server like Node.js.

3. What was the most popular front-end js library?

React.js

4. What is the most popular front-end js library today?

React.js remains the most popular front-end JavaScript library. It has been the leading choice for developers for several years due to its competent-based architecture, virtual DOM for efficient updates, and strong ecosystem with tools like Redux for state management.

Engagement Questions (added)

1. What is AJAX?

AJAX (Asynchronous JavaScript and XML) is a technique used in web development to send and retrieve data from a server asynchronously, without reloading the entire web page.

2. Why do you put your JS Code in separate file?

Putting JS code in a separate file makes it reusable for other web pages. It also helps organizing the code, rather than having them in an HTML file.
