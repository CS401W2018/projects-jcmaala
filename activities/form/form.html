Engagement Questions:

1. The form tag has an "action" attribute. What does it do?

The action attribute in HTML specifies the URL where the form data should be sent when the user submits the form.

2. The form tag has a "method" attribute. What does it do?

The method attribute in HTML specifies how the form data should be sent to the server when the form is submitted.

3. What is the difference between a "POST" and a "GET"?

The "GET" is used for retrieving or filtering data like search forms, links, etc. while "POST" is used for sensitive data or actions that modify data like login, registration, and form submissions.
