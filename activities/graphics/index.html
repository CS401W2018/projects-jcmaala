Questions:

1. What kind of photos are good to encode using JPEG?

Photographs, social media banners, web banners and backgrounds are just some of the photos that fits good using JPEG in HTML. The reason being is that JPEG is a great format for HTML because of its compression and compatibilty.

2. What kind of photos are good to encode using GIF?

Some examples of photos that are good to encode using GIF are animated images, small icons and UI elements, and even simple graphics with just some few colors.

3. What is unique about the PNG format?

One of the factors that makes PNG unique is that it is widely supported in web browsers and HTML. This makes it a standard choice for high-quality images in HTML.

4. How do you resize your photo to the size it should be displayed?

You can resize the photo using CSS setting the width and height of the photo.

5. Why should you resize photos using photo editing software instead of resizing it using CSS?

Because it reduces the file size and loading time of the photo. This also maintains the image quality of the photo.
