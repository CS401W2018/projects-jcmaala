Engagement Questions:

Question: Explain the difference between an absolute link and relative link. Why would you use relative links?

Answer: Absolute links includes complete URL with the domain name. For example https://facebook.com while relative link provides path that are related to the location of the current file. For example ../page/mypage.html

Question: List the 3 rules with Relative Links

Answer: Same folder. For example file.html. Another one is Child folder. For example assets/styles.css. And last one, UP. For example ../file/page.html.

Question: Do links always need to be inside the NAV tag?

Answer: No, links do not always need to be inside the NAV tag.
