Engagement Questions:

1. Explain the Difference between inline, embedded, and external CSS.

Inline CSS is written inside an HTML tag with a "style" attribute. Embedded CSS is written inside the HTML file using the style tag. And lastly, external CSS is written in a separated CSS file, used by HTML through using link tag.

2. Explain why you should only use external CSS.

Using external CSS allows you to only use one CSS file if for example you are developing a website with multiple pages. With using external CSS, it's easier to add or delete a specific website design without ruining the whole page. Plus, external CSS also allows other people you collaborated with to make easy additions to the design of the website.

3. What is a selector used for?

A selector in CSS is used to target a specific part of the website that needs tweaking or needs a specific design. Using selector helps the developer to target specific HTML elements so that it's easier to apply design to them as well.
