Engagement Questions:

1. What is normal flow in HTML?

Normal flow refers to the default style and behavior of the HTML elements before the CSS positioning like flex, float, or grid are applied.

2. What does float do? Why do you need to clear the float?

The float is used to move an element like img to the left or right of the page while allowing other content to wrap around it. You need to clear the float because it removes the element from the normal document flow.

3. What do absolute, fixed, and relative do?

The relative does not remove the element from the normal flow. The absolute is removed from the normal flow and positioned to the nearest positioned ancestor. Fixed does not move when you scroll the webpage.

4. How can you make nice navigation buttons for mobile devices?

Creating a nice navigation buttons for mobile devices comes with touchable size. One thing that you can do is to set a padding and borders so it is more bigger and easier to click.
