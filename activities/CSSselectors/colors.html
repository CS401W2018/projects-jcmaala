Answer Questions:

1. List out different ways to select an item.

You can select an item in CSS through Tag Name, Decendant Rule, ID, and Class.

2. What is the difference between and ID and a Class?

ID is used for a specific tag or part of the HTML that the developer wants to edit. This cannot be reused unlike Class that can be reusable by other HTML tags and can be assigned multiple times in the HTML.

3. What is the difference between a SPAN tag and a DIV tag?

SPAN tag is used to specifically target a part of the HTML that needs editing, without really affecting other parts of the HTML. On the other hand, DIV is used to divide a part or group in the HTML to edit those specific group in the CSS.

4. What is the difference between DIV tag and a HEADER, MAIN, or FOOTER?

DIV tag is used for dividing specific parts of the HTML for better and easier styling in CSS, HEADER is used for HTML parts like the navigation, menus, and logos of the webpage, MAIN is where most of the contents of your website goes, and lastly FOOTER is where you put the copyright information of the webpage, contact details, and other links related to the webpage content.

5. What are the different ways to select a color?

You can select a color through its name, through RGB, hex code, short hex code, and lastly, the Hue Saturation and Value (HSV).
