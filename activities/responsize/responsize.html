Engagement Questions:

1. Why is the viewport meta tag importan?

Because it controls how a webpage is displayed on different screen sizes, particularly on mobile device.

2. How can you adjust the CSS for printing?

You can make a new CSS file for printing specifically. To do so, you can do the usual syntax in linking the CSS from your HTML, and then adding the media="print" syntax.

3. When would you want to adjust the CSS for printing?

You would want to adjust it in ensuring that the fonts are readable and that the unnecessary elements like navigation are not included in the printed handout.
