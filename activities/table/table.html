Engagement Questions:

1. When do you use a table?

You need to use a table when you are presenting a tabular data, or information in your website that is best to be displayed in rows and columns.

2. When do you not use a table?

You should not use a table for page layout or website design, if you are using just the simple list, and for forms and input fields.

3. When do you need to do extra work for table accesibility?

You need extra work for table and accessibility when creating complex tables to ensure that users with screen readers can understand the table's structure and meaning.

Engagement Questions (added)

1. Why don't we use the border attribute?

The border attribute in HTML tables is considered depricated in modern web development because it uses presentational markup instead of separating content from style.

2. What is the easiest way to color every other row a different color?

The easiest way to color every other row in an HTML table is by using the :nth child(even) or :nth child(odd) pseudo-class in CSS.
